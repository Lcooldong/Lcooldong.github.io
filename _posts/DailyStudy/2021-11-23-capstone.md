## Interrupt
GPIOG->ODR = GPIOG->ODR ^ (1 << 10);  
원하는 부분 반전
<br>
ODR 레지스터 : OUT으로 설정된 PIN의 출력여부 결정하는 레지스터 (0~15)  
Output data register  

{% highlight html %}

{
  if(GPIO_Pin == GPIO_PIN_15)
  {
	GPIOG->ODR = GPIOG->ODR ^ (1 << 10);
  } 

}
