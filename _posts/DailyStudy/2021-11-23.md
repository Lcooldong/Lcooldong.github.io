## Interrupt
GPIOG->ODR = GPIOG->ODR ^ (1 << 10);  
원하는 부분 반전
<br>
ODR 레지스터 : OUT으로 설정된 PIN의 출력여부 결정하는 레지스터 (0~15)

{% highlight html %}
void HAL_GPIO_EXTI_Callback(uint16_t GPIO_Pin)
{
  if(GPIO_Pin == GPIO_PIN_15)
  {
	GPIOG->ODR = GPIOG->ODR ^ (1 << 10);
  } 

}
{% endhighlight %}
<br>
G포트 10번 반전,  15번 출력핀(SW7)으로 인터럽트 생성
