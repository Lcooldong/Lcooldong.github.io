## Install <br>

- Nodejs download : <https://nodejs.org/ko/download/releases/> <br>

### Open your CMD <br> 

- Here is how to install node-red ▼
{% highlight html %}
{% raw %}
npm install -g --unsafe-perm node-red
{% endraw %}
{% endhighlight %}
<br>
- This is execution command. ▼
{% highlight html %}
{% raw %}
node-red
{% endraw %}
{% endhighlight %}
<br>
- You can use node-red on Web browser. ▼
{% highlight html %}
127.0.0.1:1883 or 192.168.0.X (Local network)
{% endhighlight %}
<br>





###  Additional pallete on Node-red

node-red-dashboard
